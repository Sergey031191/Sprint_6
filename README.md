Финальный преокт 6го спринта курса Атоматизатор Тестирования.

Структура проекта:
Папка data - в ней хранятся файлы с константами
Папка locators - в ней хранятся локаторы к страницам
Папка pages - в ней хранятся веб-страницы в виде POM
Папка tests - в ней хранятся файлы с тестами
Папка allure_results - allure результаты прогона тестов
